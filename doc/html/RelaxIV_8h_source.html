00421 
00422    - kCLP is the "LP" format read by several LP solvers;
00423 
00424    - kRIV is the following RelaxIV-specific format:
00425 
00426           - < number of nodes > < number of arcs >
00427 
00428           - for( < each arc > )
00429             < start node > < end node > < reduced_capacity > < reduced_cost >
00430 
00431           - for( < each node > )
00432             < reduced flow deficit at node >
00433 
00434           \note the data of the problem in this format is not that of the
00435                 original problem, but rather that of the "reduced" problem
